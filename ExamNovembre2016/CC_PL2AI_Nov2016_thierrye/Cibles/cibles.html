Dessiner des cibles

Cette page présente l'utilisation d'une fonction javascript permettant de dessiner des cibles

Exemple 1

Dans le canvas ci-dessous, trois cible sont dessinées.

  • cible 1: centre (x:100, y:150), rayon externe 75, rayon interne 10, nombre de cercles 5
  • cible 2: centre (x:325, y:150), rayon externe 125, rayon interne 10, nombre de cercles 8
  • cible 3: centre (x:625, y:150), rayon externe 150, rayon interne 10, nombre de cercles 15
le canvas où seront dessinées 3 cibles

Exemple 2

Dans les deux canvas ci-dessous, deux pavages de cibles sont effectués.

  • Dans le canvas de gauche, un pavage complet avec 5 cibles par ligne, chaque cible ayant un rayon interne de 5 pixel et un nombre de cercles de 7.
  • Dans le canvas de droite, le même pavage mais alterné (une cible sur deux est dessinée).
