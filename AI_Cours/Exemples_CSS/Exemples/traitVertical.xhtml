L'objectif de ce TP est de mettre en oeuvre les principales balises HTML. Le but de ce TP est également de vous ``laisser'' rechercher les informations nécessaires sur les sources disponibles (internet essentiellement).

Créez un fichier html portant votre nom. Ce fichier sera constitué des éléments suivant :

  • Une partie entête. Le titre du fichier est votre nom
  • Une partie corps. La couleur de fond de la page est le blanc
  • Un premier titre centré (balise <H1>..</H1>) : ``CV de votre-nom''.
